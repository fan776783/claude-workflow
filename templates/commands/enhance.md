# Prompt 增强

将简单的用户输入转化为结构化、详细的提示词，提升 AI 响应质量。

---

## Usage

```bash
/enhance "<原始提示词>"
```

---

## 执行流程

### 1. 检测增强服务
- 检查 `mcp__auggie-mcp__prompt-enhancer` 是否可用
- 不可用时使用内置增强逻辑

### 2. 调用增强服务
- 将原始输入发送给增强服务
- 获取结构化的增强版本

### 3. 展示对比
```
📝 **原始输入**
[用户输入]

✨ **增强版本**
[增强后的详细提示词]
```

### 4. Hard Stop 等待确认
```
**是否使用增强版本？(Y/N/M)**
- Y: 使用增强版本继续
- N: 使用原始版本
- M: 手动修改后使用
```

## 注意事项

- **Hard Stop**：必须等待用户确认后才能继续
- **保留原意**：增强不改变原始需求的核心意图
- **适度补充**：避免过度假设用户需求
- **可选使用**：用户可以选择使用原始版本
